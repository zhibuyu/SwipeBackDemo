# SwipeBackDemo
侧滑返回，支持左滑、右滑、上滑返回，兼容Viewpager

注意在manifest文件中定义theme，添加如下属性：

    <style name="AppTheme.NoActionBar">
       <item name="android:windowIsTranslucent">true</item>
    </style>
